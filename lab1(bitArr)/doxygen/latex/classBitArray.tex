\doxysubsection{Detailed Description}
a dynamic bit array for useful storage. 

The \mbox{\hyperlink{classBitArray}{Bit\+Array}} class supports bitwise operations, dynamic resizing, and other useful methods for working with binary data. The class works with unsigned long sized memory blocks.
